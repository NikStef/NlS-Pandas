# Netflix Data Cleaning (Pandas)

This project focuses on cleaning and preparing the Netflix titles dataset using pandas.

Key steps include handling missing values, removing duplicates, converting data types, and extracting useful features (such as movie duration and year added) to prepare the data for analysis.
